### **Graph Construction (Section 9)**

**Hyperparameters:**
- `k = 10`: Number of nearest neighbors per node
  - Rationale: Balances sparsity vs. over-smoothing
  - Ablation candidates: {5, 10, 15, 20}

- `alpha = 0.00135`: Temporal decay rate
  - Half-life: ~512 days (ln(2) / 0.00135)
  - Ablation candidates: {0.0001, 0.0005, 0.00135, 0.005}

**Edge Construction:**
1. Semantic edges: k-NN based on cosine similarity of BERT embeddings
```
distance = cosine_distance(emb_i, emb_j)
sim = 1 - distance ∈ [0, 1]
```

2. Temporal weighting applied to edges:
```
dt_days = |timestamp_i - timestamp_j| / 86400
temporal_decay = exp(-α × dt_days)
final_weight = semantic_sim × temporal_decay
```

**Graph Statistics:**
- Nodes: `N_balanced`
- Edges: `k × N_balanced` approximately (directed star topology)

**Advanced Graph Options (Section 9.5) - Utilities provided but optional:**
- Temporal bin edges: Connect news within same time window (e.g., weekly)
- Category-aware edges: Within-category k-NN
- Combined graph merging: Weighted fusion of multiple topologies

- Architecture:

  2. **Temporal branch**: Learns edge time embeddings separately
  3. **Fusion**: Time embeddings passed to GAT as edge attributes

- Structure:
